REA App - Manage Google Drive Permissions

The REA App is a tool that helps Google Drive users manage file permissions easily and securely.

Key Features:

  • List users who have access to your Google Drive files.
  • Search and view which files a specific user has access to.
  • Remove a user's access to specific files.

The app does not store or share any user data and performs actions in real-time using the Google Drive API.
